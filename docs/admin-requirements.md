- [ ] 발송 실패 건 재시도 큐

**데이터:**
- 읽기: `applications` (status=approved), `cohorts.approved_sms_template`, `cohorts.latpeed_payment_url`

```
pending → approved → (SMS 발송) → payment_sent → (결제 완료) → paid → (매칭 후) → matched
  ↓
rejected (종결)
```

## 4. 비기능 요구사항
- **응답 속도:** 리스트 로딩 < 1초 (100건 이하 가정)
- **동시성:** v1은 호스트 1명이라 낙관적 락 정도면 충분
- **백업:** Supabase 자동 백업 (무료 플랜 기본)
- **프로덕션 배포:** Vercel (어드민은 비공개 URL 또는 Basic Auth 추천)
